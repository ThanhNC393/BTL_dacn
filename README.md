**Student Result Management System (SRMS)** is a web-based platform designed to help **students and lecturers** manage and track academic performance efficiently.

- **Lecturers** to input, update, and manage student grades.
- **Students** to view their grades and monitor academic progress.
- **Administrators** to manage accounts, subjects, and roles.
